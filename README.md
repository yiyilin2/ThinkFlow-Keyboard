# ThinkFlow Keyboard 🚀

ThinkFlow is an innovative Android Input Method (IME) designed to help language learners "Think in English" by translating in real-time.

---

### 🤖 An AI-Collaborative Project

ThinkFlow is proud to be an **AI-Collaborative Project**. The entire keyboard's architecture, complex translation logic, and UX optimizations were co-created and debugged through iterative collaboration between the human developer and an advanced Large Language Model (LLM). 

---

## ✨ Key Features
- **Real-time English-to-Chinese Translation**: Powered by Google ML Kit (Full Offline Support).
- **Localized processing**: 100% processing occurs on your device, ensuring maximum privacy. No data is collected or uploaded.

## 🛠️ Tech Stack
- **Language**: Java
- **AI Engine**: Google ML Kit (Translate API)
- **Framework**: Android SDK (InputMethodService, CardView, Materials)

## 📥 How to use
1. Install the APK from the [Releases page](https://github.com/yiyilin2/LanguageKeyboard/releases) 
2. Enable "ThinkFlow" in System Settings.
3. Switch current keyboard to ThinkFlow and start thinking!
